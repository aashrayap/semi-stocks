# EUV Tools

EUV lithography is the hard physical ceiling at the end of the cascade. If AI demand keeps compounding, eventually the constraint is not wafers or memory but the number of advanced lithography tools that can be built.

## Why It Matters

- ASML is effectively the only EUV supplier
- Zeiss optics remain the true sub-constraint underneath ASML's output
- `data/thesis.yaml` tracks an approximate ceiling of 100 EUV tools per year by 2030
- The same file tracks 3.5 EUV tools per gigawatt of fab capacity

## Current Read

This stage is still `next`, not active. The repo treats ASML as a watchlist name where commentary on bookings, lead times, and capacity is more important than current quarter revenue alone.
